# "Tools for Life" Button Design Options

Here are 5 minimalistic but attention-grabbing button design options:

## Option 1: Subtle Pulse Glow (Recommended)
- White text on transparent background
- Thin white border
- Subtle pulsing glow animation
- Hover: border brightens, slight scale up
- **Class**: `text-white bg-transparent border border-white/30 hover:border-white hover:scale-105 btn-pulse`

## Option 2: Animated Border
- White text, transparent background
- Border that subtly pulses/breathes
- Hover: fills with white background
- **Class**: `text-white bg-transparent border-2 border-white/40 hover:bg-white hover:text-black btn-border-animate`

## Option 3: Soft Shadow Float
- White text, transparent background
- Soft shadow that creates depth
- Subtle floating animation
- Hover: shadow intensifies, lifts up
- **Class**: `text-white bg-transparent border border-white/20 shadow-lg shadow-white/10 hover:shadow-white/30 hover:-translate-y-1 btn-float`

## Option 4: Minimal Outline Fill
- White text, transparent background
- Clean outline
- On hover: fills with white, text turns black
- Smooth transition
- **Class**: `text-white bg-transparent border-2 border-white/50 hover:bg-white hover:text-black transition-all duration-300`

## Option 5: Gradient Accent
- White text
- Subtle gradient border (white to gray)
- Hover: gradient intensifies
- **Class**: `text-white bg-transparent border border-transparent bg-gradient-to-r from-white/20 to-white/5 hover:from-white/40 hover:to-white/20 backdrop-blur-sm`

---

**Current Implementation**: Option 1 (Subtle Pulse Glow)
