# Correct Steps for GitHub Desktop

## The Right Way:

### Step 1: Create a New Repository (Not Add Local)
1. In GitHub Desktop, click **"File"** → **"New repository"** (NOT "Add Local Repository")
2. Fill in:
   - **Name**: `retirement-calculator`
   - **Description**: "Retirement Corpus Calculator" (optional)
   - **Local path**: Click "Choose" and select: `C:\Users\shaur\Downloads\VG cursor`
   - **Git ignore**: Select "Node" from the dropdown
   - **License**: None (or choose one)
3. Click **"Create repository"**

This will:
- Initialize Git in your folder
- Connect it to GitHub
- Set everything up automatically

### Step 2: Commit Your Files
1. You'll see all your files listed
2. At the bottom, type: "Initial commit - Retirement Corpus Calculator"
3. Click **"Commit to main"**

### Step 3: Publish
1. Click **"Publish repository"** (top right)
2. Make sure it's set to **Public** (uncheck "Keep this code private")
3. Click **"Publish repository"**

---

## Alternative: If "New repository" doesn't work

Try this in GitHub Desktop:
1. Click **"File"** → **"New repository"**
2. Or click the **"+"** button (top left) → **"Create New Repository"**
